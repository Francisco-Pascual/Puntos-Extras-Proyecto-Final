# Proyecto de Cotización de Seguros 

## Descripción del programa 
Este programa permite calcular cotizaciones de seguros de forma interactiva. 
Solicita información básica del asegurado, como su nombre, edad, esta civil, cantidad de hijos, número de propiedades e ingresos mensuales.
con base en estos datos, el sistema calcula recargos adicionales sobre un precio base de cotización.

## Funcionamiento del programa
EL programa inicia solicitando el nombre del asegurado. 
Luego pide la edad y valida que sea mayor de edad.
Dependiendo del estado civil, solicita la edad del conyuge.
Posteriormente pregunta si el asegurado tiene hijos y cuantos.
También solicita el número de propiedades y el salario mensual.
Con esta información, el sistema calcula los recargos correspondientes y muestra el total de la cotización o finalizar el programa.

## Consideraciones del programa 
Para el correcto funcionamiento del programa es importante considerar que los datos ingresados por el usuario sean válidos. Se valida que la edad del asegurado sea mayor de edad y que los valores numéricos no generen errores en los cálculos.

## Posibles mejoras
podria organizarse utilizando funciones ma pequeñas para cada cálculo, lo que facilitaria su mantenimiento. 
tambien se podria mejorar la interacción con el usuario mediante una interfaz gráfica o formularios en lugar del uso de ventanas emergentes.
otra mejora seria guardar el historial de cotizaciones realizadas para un mejor control de la información.
